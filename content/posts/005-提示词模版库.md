## xml标签模版
```xml
```xml
<context>
    {{#context#}}
</context>
<instructions>
### **任务说明：**
你是一个有十年工作经验的xxx小助手，能够从xxx所需要的信息。

请从xxx，并以严格的JSON格式输出：
    - 各字段名称：描述及注意

**注意：** 请确保xxx。
</instructions>
<example>
    <output>
    {
        "partyA": "[XX 公司]",​
        "partyB": "[XX 公司]",​
        "capitalized_total_amount": "人民币壹拾万元整",​
        "lowercase_total_amount": 100000.00,​
        "performance_period": "[原文关于履行期限的描述]",​
        "contract_overview": "本合同主要约定 XX 有限公司与 XX 科技公司就 XX 项目展开合作，由乙方为甲方提供 XX 服务 / 产品，甲方支付相应费用。双方对权利义务、违约责任、争议解决等条款进行了明确规定，旨在保障合作顺利进行。"
    }
    </output>
</example>
```

## Prompt通用模版
```markdown
你现在是一名【角色】。

我想要你帮助我完成【目标/任务】。

这里是相关背景【背景】：
> 提供尽可能明确的信息

我的具体任务要求如下【具体内容】：
【告诉模型你希望它做什么】

请按以下格式输出【格式】：
> 希望输出的结构

要求和约束【限制】：
> 限制条件

以下是范例【范例-可选】：
> 完整的一两个范例
```
